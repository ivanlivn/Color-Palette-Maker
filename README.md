# Color Palette Maker

A web app built with React to create and save color palettes.

Color steps generated by chroma.js

## Getting Started

### Installing

To install on your own system, simply clone the repo and run the following command:

```
npm init
```

## Built With

- React
- React-Router
- Material-UI
- Material-UI/styles
